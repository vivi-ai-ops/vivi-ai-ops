# Text Analyzer

This Javacript function take in a string and returns an object containing the following information:

-Number of characters
-Number of Words
-Number of unique words
-Most frequent word(s)
-Average word length

#Example

analyzeText("Hello world! Hello JS world.");
Output: { characters: 21, words: 5, uniqueWords: 4, mostFrequent: ["hello","world"], averageWordLength: 4.2 }
